# Progetto Tic Tac Toe

Autore: **WeiSheng Chen**  
Classe: **4ID**

## Descrizione
Questo progetto mostra come il gioco del **Tris (Tic Tac Toe)** possa essere analizzato. 
Il programma è scritto in **Java** e utilizza una struttura ad **albero** per rappresentare tutte le mosse possibili dei giocatori.


## Funzionamento
- I due giocatori sono:
MAX (il giocatore), che gioca con la X;
MIN (il computer), che gioca con la O.
- Il computer utilizza l’**algoritmo Minimax** per valutare le mosse migliori.
- Dopo ogni turno, il programma suggerisce la **mossa ottimale** per vincere o pareggiare.


## Struttura del progetto
- `Main.java` → Avvio del gioco  
- `Game.java` → Gestione della logica del Tris  
- `Board.java` → Rappresentazione della griglia  
- `GameTree.java` → Implementazione dell’albero delle mosse  
- `style.css`, `index.html`, `design.html` sonoe le presentazioni del progetto
